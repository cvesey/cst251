Challenge

  • The images weren't aligning side-by-side so we added "display: flex;" to the CSS and that worked.

  • We added CSS to change the links so that they weren't underlined and hade black text. The code only responded when the screen width was below 600px, and goes back to the default when above 600px. This was because we put the media query above that tag. By putting the media query tag at the bottwem and adding ""!important" it didn't affect the other tags.

Problems

  • We wanted the nav bar to be all the way to the right and then get get closer and ajust as the widow decreased in size. We found this some code on w3schools (i.e., https://www.w3schools.com/howto/howto_css_topnav_right.asp) but weren't able to adapt it to our code.

  • Shrinking the width of the screen to 600px didn't change the image layout. The navbar and header changed just not the images.
